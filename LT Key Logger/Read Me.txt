							         ** DESCRIPTION **

---THE KEY STROKES ARE LOGGED, AND SAVED IN A .TXT FILE LOCATED IN THE SAME DIRECTORY AS THE APPLICATION. (ex: if the "lt key logger" is located on your desktop then the .txt file will pop up on your desktop)
